# Tic Tac Toe 🎮

A simple **Tic Tac Toe game** built using HTML, CSS, and JavaScript.  
Play against a friend in this fun two-player game!

## ✨ Features
- Two-player mode
- Win detection (rows, columns, diagonals)
- Reset button to play again
- Simple and responsive UI
